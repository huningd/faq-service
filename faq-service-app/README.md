# faq-service-app
## Install
```shell
npm install
bower install
tsd install
grunt
```

## Hints
add new tools with:
```Bash
npm install --save grunt
```

add new dependencies with:
```shell
bower install --save alertifyjs
```
Add new type definitions with:
```shell
tsd install alertify --save
```
Query for type definitions with:
```shell
tsd query alertify
```
